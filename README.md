# Ai-Auto
bp插件，借助Ai分析数据包
目前主要实现的功能:  
1.右键发送request和response到Ai去分析。  
2.内置了一些prompt。  
3.目前支持deepseek和qwen。  
4.分析界面进行了优化，发送过来的数据包会进行排序，同时request和response也会展示  
5..增加了多轮会话，可以接着Ai回答的答案继续提问。  
6.其他功能正在思考中🤔
